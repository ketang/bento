# Build vs Buy

Use this skill before building a substantial feature, component, or subsystem
unless the user explicitly says to build from scratch.

## Workflow

1. Search for viable open-source libraries or hosted services.
2. Narrow to the top candidates based on fit, maintenance, adoption, and cost
   where relevant.
3. Present a comparison table that includes:
   - license
   - pros
   - cons or risks
   - fit for this use case
4. Ask which option to pursue, including the option to build from scratch.

## Skip Conditions

Skip this workflow only when the user explicitly says:

- build from scratch
- no library search needed
